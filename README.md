# Huawei HCCDA-Tech Essentials Learning Guide

เอกสารนี้เป็นหน้าเริ่มต้นสำหรับการเรียนและเตรียมสอบใบเซอร์ `Huawei HCCDA-Tech Essentials` ในรีโปนี้

## เป้าหมายของรีโปนี้

- ใช้เป็นโน้ตสรุปบทเรียนสำหรับทบทวนก่อนสอบ
- ช่วยจัดลำดับการอ่านจากพื้นฐานไปจนถึงจุดที่ออกสอบบ่อย
- รวมเอกสารหลักและไฟล์ประกอบให้อยู่ในที่เดียว

## รายละเอียดอย่างย่อ (บทเรียนมีอะไรบ้าง)

- `Chapter 1: Diving into Huawei Cloud` พื้นฐาน Cloud, Service/Deployment Model, Region และ AZ
- `Chapter 2: Compute Services` บริการคอมพิวต์หลัก เช่น ECS, image, scaling และแนวคิดการเลือกใช้งาน
- `Chapter 3: Storage Services` บริการจัดเก็บข้อมูลแบบ block/file/object และแนวทางเลือกให้เหมาะกับงาน
- `Chapter 4: Networking Services` โครงสร้างเครือข่ายบนคลาวด์ เช่น VPC, subnet, EIP, NAT, ELB
- `Chapter 5: Database Services` บริการฐานข้อมูลบน Huawei Cloud และแนวทางใช้งานเบื้องต้น
- `Chapter 6: Security Services` IAM, การเข้ารหัส, การบันทึกและตรวจสอบกิจกรรมด้านความปลอดภัย
- `Chapter 7: Elastic Cloud Services for Distributed Deployment` การออกแบบระบบกระจายตัวและยืดหยุ่น
- `Chapter 8: Cloud Native and Transformation` แนวคิด Cloud Native, container, และการทรานส์ฟอร์มระบบ
- `Chapter 9: Exam Outline and Sample Questions` โครงสร้างข้อสอบและข้อคำถามตัวอย่างสำหรับซ้อมก่อนสอบ
