# HCCDA

เอกสารสรุปสำหรับโปรเจกต์ **HCCDA** เพื่อให้ผู้ใช้งานใหม่กดเข้ามาแล้วเริ่มเรียนรู้ได้ทันที

## เนื้อหาหลักในรีโป

- เอกสารสรุปหลัก: `HCCDA_HCCDA_Summary_HEADING_FIXED_v2.md`
- โฟลเดอร์รูปภาพ/ไฟล์ประกอบ: `assets/`

## วิธีเริ่มอ่าน

1. เปิดไฟล์ `HCCDA_HCCDA_Summary_HEADING_FIXED_v2.md`
2. อ่านตามหัวข้อหลักจากบนลงล่าง
3. เมื่อเจอการอ้างอิงรูปหรือสื่อประกอบ ให้เปิดจากโฟลเดอร์ `assets/`

## โครงสร้างโฟลเดอร์ (ภาพรวม)

## เหมาะกับใคร

- ผู้ที่เพิ่งเข้ามาในรีโปนี้และต้องการเริ่มเรียนรู้แบบรวดเร็ว
